# Jumping_Robot
---
## V1版本

### 一、更改模型内容

**1. 优化了模型**：
本次修改优化了模型，减少了穿模的可能性。并且尽量地达到可以进行3D打印水准。

**2. 增加了齿轮**：
本次增加了齿轮，使模型更加合理。

### 二、注意事项
文件需要使用**solidworks**打开，**salto.SLDASM**文件是装配体文件，点击即可。

## V2版本
### 一、更改模型内容

**1. 扩大孔径**
本次扩大了部分孔径大小以便添加轴承。

**2. 优化模型**
减少了穿模的可能性，并且对部分零件增加厚度以便达到3D打印水准。
